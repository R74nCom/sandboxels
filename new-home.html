< A New Home

Sandboxels is moving to Neal.fun, the home of The Password Game and Infinite Craft.

We know that this might be surprising to some people, so we hope this page will answer any questions you might have.

Where are my saves?

Don't worry!! While save slots can't be automatically transferred to the new page, they are still downloadable using our new Data Transfer Page. Once downloaded, you can easily load them into your game with the "Load File" button in the Saves menu.

This page can also be used to transfer your mods and settings.

What will happen to development?

Again - don't worry! We're not going anywhere. We will continue to develop Sandboxels for both the web and Steam.

We are committed to make this transition as smooth as possible.

Where can I play Sandboxels?

When visiting the original link, sandboxels.R74n.com, you'll be automatically redirected to the new page at Neal.fun/sandboxels. This has all the same features and an improved look!

The Steam Edition will also continue to be updated.

What will happen to modding?

If you create mods for Sandboxels, there are a couple things to know. First of all, modding will continue to be supported forever.

Instead of the original R74nCom/sandboxels repository, mods will now be submitted to R74nCom/Sandboxels-Mods. All existing mods have been moved over.

For testing purposes, Sandboxels will still load mods from the "mods/" folder of the game directory when using a local file version of the game. On the actual website, mods will be loaded from the new repository.
